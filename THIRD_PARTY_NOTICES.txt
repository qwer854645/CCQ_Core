Third-Party Gun Packs
---------------------

CCQ Default Gun Pack (ccq_default_gun)
--------------------------------------

  NOT embedded in ccq_core.

  If the player already has a TACZ default gun pack under .minecraft/tacz/
  (for example tacz_default_gun/ or tacz_default_gun*.zip), ccq_core may copy
  a whitelisted subset of files at runtime, write CCQ metadata, and output the
  result as ccq_default_gun/. The original tacz_default_gun* pack is removed
  afterward to avoid loading duplicate default packs.

  If no local default pack exists, ccq_core may extract the required files
  directly from the installed tacz mod jar
  (assets/tacz/custom/tacz_default_gun/) before writing ccq_default_gun/.

  Retained assets are derived from the TACZ Default Gun Pack (TACZ Dev Team,
  CC BY-NC-ND 4.0), including 12 gauge ammo and shared resources required by
  third-party gun packs.

Applied Armorer (TaCZ gun pack)
-------------------------------

  Applied Armorer is NOT bundled in ccq_core.

  If the player already has an Applied Armorer pack under .minecraft/tacz/,
  ccq_core may generate CCQ workbench tab configuration at runtime and write it
  into the existing pack file (in-place). No separate gun-pack patch files are
  shipped in the mod jar.

  Applied Armorer remains CC BY-NC-ND 4.0 (Koei, Ithaqua).
  https://space.bilibili.com/407541278

  Contact the authors before redistributing modified pack files.
